# Guía de inicio

## Install
Abir una terminal en la raiz del proyecto y ejecutar el siguiente comando para instalar cypress.

```
$ yarn install
```

## Ejecución de cypress
Para abir cypress ejecute el siguiente comando

```
$ yarn run cypress open
```
